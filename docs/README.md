# Общие описания решений
## area
Возвращает площадь фигуры. Рассчёт площади происходит по соответствующим вормулам:
- Окружность: S = πR²
- Прямоугольник: S = ab
- Квадрат: S = a²
- Треугольник: S = ah/2

## perimeter
Возвращает периметр фигуры. Рассчёт периметра происходит по соответствующим вормулам:
- Окружность: P = 2πR
- Прямоугольник: P = 2a + 2b
- Квадрат: P = 4a
- Треугольник: P = a + b + c

# Подробные описания
## area
### circle
- Принимает на вход одно число - длину радиуса окружности
- Рассчитывает площадь окружности по формуле: S = πR²
- Возвращает одно число - площадь окружности  
```
x = area(2)
print(x) #12.56637061
```

### rectangle
- Принимает на вход два числа - длины соседних сторон прямоугольника
- Рассчитывает площадь прямоугольника по формуле: S = ab
- Возвращает одно число - площадь прямоугольника  
```
x = area(2, 3)
print(x) #6
```

### square
- Принимает на вход одно число - длину стороны квадрата
- Рассчитывает площадь квадрата по формуле: S = 4a
- Возвращает одно число - площадь квадрата  
```
x = area(2)
print(x) #4
```

### triangle
- Принимает на вход два числа - длины основания и высоты треугольника
- Рассчитывает площадь треугольника по формуле: S = ah/2
- Возвращает одно число - площадь треугольника  
```
x = area(2, 3)
print(x) #3
```
